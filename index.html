Recent Projects.

Here you can find some of the projects that I created recently

HommieSpace

HommieSpace is a property technology startup helping people rent out available rooms in a shared home. Built using React, Django, Restframework, Redis, PostgreSQL, WebSockets, etc.

PayLoadNg

PayLoadNg is a web app that helps individual easily save little by little towards their goals. It was a great experience to build the whole project with Bootstrap, Vanilla Js, Django, PostgreSQL.

Virtual School(LMS)

This is a full-fledged online school. It covers advanced features like real-time communication(chat),progress tracking, certificate generation, Quiz, Assigments, etc. Made using React, Django, RestFrameWork, Celery, WebSockets, PostgreSQL.

iTalently

iTalently is an employment and recruitment web project I created. It helps people search hundreds of jobs online to find the next step in their careers. It is also used by HR to receive applications. Made using Html, Css, Bootstrap, Javascript, Django, Celery, etc.

Secure Note App

Secure Note is a web application that allows users to share a confidential note via a web link that will self-destruct after it is read by the intended recipient. Made using Html, Css,Javascript, AOS library, Django, PostgreSQL, etc.

Rewardful

Rewardful is an online fundraising platform. It allows content creators to set up their pages and get one-time donations and memberships from fans. Tech Stack includes Bootstrap,Javascript,Django, etc.
